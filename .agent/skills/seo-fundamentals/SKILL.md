# SEO Temelleri

> Arama motoru görünürlüğü için prensipler.

---

## 1. E-E-A-T Çerçevesi

| Prensip | Sinyaller |
|-----------|---------|
| **Deneyim (Experience)** | İlk elden bilgi, gerçek örnekler |
| **Uzmanlık (Expertise)** | Kimlik bilgileri, bilgi derinliği |
| **Otorite (Authoritativeness)** | Backlinkler, bahsetmeler, endüstri tanınırlığı |
| **Güvenilirlik (Trustworthiness)** | HTTPS, şeffaflık, doğru bilgi |

---

## 2. Temel Web Verileri (Core Web Vitals)

| Metrik | Hedef | Neyi Ölçer |
|--------|--------|----------|
| **LCP** | < 2.5s | Yükleme performansı |
| **INP** | < 200ms | Etkileşim |
| **CLS** | < 0.1 | Görsel kararlılık |

---

## 3. Teknik SEO Prensipleri

### Site Yapısı

| Öğe | Amaç |
|---------|---------|
| XML site haritası | Taramaya yardımcı ol |
| robots.txt | Erişimi kontrol et |
| Canonical etiketleri | Kopyaları önle |
| HTTPS | Güvenlik sinyali |

### Performans

| Faktör | Etki |
|--------|--------|
| Sayfa hızı | Temel Web Verisi |
| Mobil uyumlu | Sıralama faktörü |
| Temiz URL'ler | Taranabilirlik |

---

## 4. İçerik SEO Prensipleri

### Sayfa Öğeleri

| Öğe | En İyi Uygulama |
|---------|---------------|
| Başlık etiketi (Title tag) | 50-60 karakter, anahtar kelime önde |
| Meta açıklama | 150-160 karakter, ilgi çekici |
| H1 | Sayfa başına bir tane, ana anahtar kelime |
| H2-H6 | Mantıksal hiyerarşi |
| Alt metni | Açıklayıcı, doldurulmuş değil |

### İçerik Kalitesi

| Faktör | Önem |
|--------|------------|
| Derinlik | Kapsamlı kapsam |
| Tazelik | Düzenli güncellemeler |
| Benzersizlik | Özgün değer |
| Okunabilirlik | Net yazım |

## 5. Şema İşaretleme Türleri

| Tür | Kullanım |

## 6. AI İçerik Yönergeleri

### Google Ne Arar

| ✅ Yap | ❌ Yapma |
|-------|----------|
| AI taslağı + insan düzenlemesi | Ham AI içeriği yayınlama |
| Özgün içgörüler ekle | Değer katmadan kopyala |
| Uzman incelemesi | Doğruluk kontrolünü (fact-check) atla |
| E-E-A-T'yi takip et | Anahtar kelime doldurma |
